This tool allows to execute various operations on uuCloud using CLI.

# How to install ?

`npm install --registry "https://repo.plus4u.net/repository/npm/" -g uucloud-cli`

# How to use ?

`uucloud --help`
`uucloud ps --help`
`uucloud logs --help` 

# How to develop ?

Publish to npmjs: 

`npm publish --registry http://registry.npmjs.com`
